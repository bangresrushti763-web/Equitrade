# Image Assets for Equitrade

This directory contains the following images:

1. `equitrade.svg` - Main logo for the Equitrade platform with full name
2. `equitrade-icon.svg` - Simplified icon version of the Equitrade logo
3. `favicon.svg` - Favicon version of the Equitrade logo
4. `equitrade.png` - PNG version of the logo for compatibility
5. `srushti.png` - Personal image of Srushti Bangre

These images are used throughout the Equitrade application.
